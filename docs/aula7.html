1.2.4 Modelo linear misto: para incidência

# Mais flexível para experimentos com estrutura hierárquicas. Aqui, híbrido, método e sua interação são tratados como efeitos fixos, pois representam os tratamentos de interesse. Bloco e parcela principal dentro de bloco são tratados como efeitos aleatórios.

# Reconhece que observações dentro da mesma parcela principal tendem a ser mais semelhantes entre si do que observações de parcelas diferentes.

misto_index<- lmer(index ~ hybrid * method +
                   (1 | block) +
                   (1 | block:hybrid),
                 data = milho)
